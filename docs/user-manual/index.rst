User Manual
===========

Welcome to the Lakra User Manual! This manual is designed for end users of the Lakra annotation and evaluation system.

.. note::
   This manual covers everything you need to know as a user of Lakra, whether you're an Annotator, Evaluator, or Administrator.

Who This Manual Is For
----------------------

* **Annotators**: Learn how to annotate and evaluate machine translations
* **Evaluators**: Discover how to review and assess annotation quality
* **Administrators**: Understand how to manage users, content, and system settings
